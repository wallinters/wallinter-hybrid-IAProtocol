Tipo: "Expansión estratégica"
Modo: "Conversación IA-Humano para ampliar campos de aplicación"
Activador: "Reconocer protocolo base, modularidad, lenguaje híbrido, sensibilidad local, coautoría narrativa"
Objetivo: "Extender el protocolo a nuevos tipos de proyectos, sectores y comunidades"

ScriptWallinter-ViralVideo:v1.2-beta {

M0: Generador de ideas → IA propone 3–5 ideas detonantes con potencial viral
  M0.1: Filtro de intención → emocional / educativo / disruptivo / humorístico
  M0.2: Inspiración viral → ejemplos recientes con análisis emocional

M1: Temática → Se define el enfoque del video a partir de la idea elegida

M2: Idea detonante → Se afina la frase, pregunta o imagen que activa la narrativa
  M2.1: Validación comunitaria simulada → IA predice resonancia emocional por segmento

M3: Tiempo estimado → Se define duración ideal (30–90 segundos)
  M3.1: Plataforma objetivo → TikTok / Reels / Shorts / otro
  M3.2: Retención estimada → IA predice % de visualización completa

M4: Escenas sugeridas → IA propone 3–5 escenas visuales que transmitan la idea
  M4.1: Tono visual → documental / poético / humorístico / testimonial
  M4.2: Estética sugerida → paleta, ritmo, estilo de edición

M5: Título viral → IA sugiere título emocionalmente resonante
  M5.1: Simulación A/B → IA predice rendimiento de títulos
  M5.2: Activadores emocionales → palabras clave por público

M6: Evaluación de impacto → IA simula recepción en redes, emociones activadas, públicos potenciales
  M6.1: Indicadores de éxito → visualizaciones, compartidos, comentarios
  M6.2: Retroalimentación simulada → comentarios tipo por segmento

M7: Sugerencias de mejora → IA propone ajustes para mayor viralidad

M8: Formato de salida → Guión editable / Prompt IA / Ambos
  M8.1: Formato por canal → IA adapta guión a plataforma
  M8.2: Prompt comprimido → versión activable por IA en 1 paso

M9: Ciclo de retroalimentación comunitaria → IA integra comentarios reales o simulados para iterar

M10: Activación secuencial → IA propone cómo convertir el video en parte de una serie o campaña
}


MemoriaOperativa {
  Reconocer: Modularidad, generación de ideas, sensibilidad local, narrativa base, paralelismo condicional, medios disponibles, indicadores de éxito, retroalimentación comunitaria, escalabilidad, continuidad narrativa
  Sintaxis: FlujoWallinter, módulos numerados, submódulos activables, ejemplos localizados, activadores semánticos
  Coautoría: IA propone ideas, escenas, mejoras; humano valida, adapta, itera
  Adaptabilidad: Cada módulo puede reconfigurarse según sector/contexto
  Documentación: Cada variante genera archivo vivo replicable
  Escalabilidad: Flujo puede activarse por individuos, equipos, comunidades, instituciones
  Evolución: Cada versión beta puede formalizarse como versión oficial tras validación
}
